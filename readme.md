# KHURSHEED GALKALA - Cyber Demo

A safe cyber-style demonstration website built with HTML, CSS and JavaScript.

## Features

- Hacker UI
- Neon Green Theme
- Responsive Design
- Demo Progress Animation
- No real API calls
- No reporting or exploit functionality

## Deploy on Vercel

1. Upload all files to a GitHub repository.
2. Log in to Vercel.
3. Import the GitHub repository.
4. Click **Deploy**.

Your website will be live in a few seconds.
